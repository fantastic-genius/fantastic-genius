# Hi, I'm Hamzah Abdulfattah 👋

## AI Engineer with a full-stack foundation

I build LLM-powered, AI-native applications and RAG systems, and I wire AI into real products. I have a strong background in multi-modal AI, AI inference, and agentic AI orchestration. My foundation is full-stack engineering, and that is what I build AI on top of. I work at the intersection of AI and engineering.

- 🧠 Focused on LLM Inference, LLM applications, RAG pipelines, Agentic AI and AI product features
- 🌍 Based in Lagos, Nigeria
- 📝 I write about production AI, cost, and governance on LinkedIn
- 🎓 Studying toward a Master's in Data Science
- 🤝 Open to collaborating on AI engineering projects

### Things I have built

- An LLM-powered customer support agent
- An AI design generation tool
- RAG pipelines and AI features shipped inside products

### What I work with

- **Languages:** JavaScript, TypeScript, Python
- **Frontend:** React, Next.js, Tailwind CSS, React Native
- **Backend:** Node.js, Express, Python, FastAPI
- **AI / ML:** LLM Inference, LLM application development, RAG, prompt engineering, AI feature integration, LLM Training/Fine Tuning
- **Data:** PostgreSQL, MySQL, MongoDB, Pinecone
- **Others:** vLLM, Unsloth, MicroServices
